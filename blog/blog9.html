Blog Update

5/03/2025

Frontend Enhancements

Over the past few weeks, our team has made solid progress in refining the UI:

  • Implemented a search bar with suggestions, making it easier to find locations.
  • Fixed several theme-related bugs.
  • Added a recenter button to quickly return to the user's current location.
  • Successfully integrated Google Maps directions for better navigation.

Implementing the grid overlay has been a huge challenge. The official documentation’s approach didn’t work as expected, making the process much more complicated than anticipated. We are now all focused on getting this done.

Backend & Feature Development

The backend side is progressing, though most updates have been focused on planning. Over the next two weeks, the team will also:

  • Finish the grid implementation and fix remaining bugs.
  • Develop a feedback system, allowing users to annotate accessibility data layers with predefined options (e.g., "wheelchair ramp," "puddle").
  • Implement the first version of text-to-speech for accessibility.
  • Set up user testing, ideally starting next week.
  • Introduce keyboard shortcuts for easier navigation.

Looking Ahead

As the project nears completion, the final three weeks will focus on:

  • Cleaning up the frontend code.
  • Improving documentation.
  • Adding more tests.
  • Integrating additional data sources.

The final week will be dedicated to wrapping up deliverables and ensuring everything is in place for a smooth launch.
